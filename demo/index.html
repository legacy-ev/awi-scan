AWI001 + AWI003

Find the AI workflow path attackers can steer.

awi-scan looks for untrusted GitHub text reaching agent prompts, then follows the next hop into shell, gh, github-script, or repository mutation steps.

Vulnerable workflow

- id: triage
  uses: anthropics/claude-code-action@v1
  with:
    prompt: |
      Triage this issue:
      ${{ github.event.issue.body }}

- run: gh issue edit "$NUMBER" \
    --add-label "${{ steps.triage.outputs.labels }}"

awi-scan report

[HIGH] AWI001 P2A
Untrusted GitHub event text reaches an AI-agent prompt

Source: issue body
Sink: anthropics/claude-code-action@v1

[CRITICAL] AWI003 P2S
AI-agent output reaches a script or GitHub mutation step

Safer rewrite:
Treat agent output as untrusted. Validate it against a strict schema and require human approval before mutations.
