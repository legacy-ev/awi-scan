# Launch Checklist

## Pre-Launch

- Replace `your-org/awi-scan` placeholders in `README.md` and SARIF metadata.
- Record a 30-second terminal GIF scanning `fixtures/unsafe-claude.yml`.
- Add screenshots from `demo/index.html`.
- Create issues for missing sink signatures and invite fixture contributions.

## Launch

- GitHub release: `v0.1.0`.
- Post a concise writeup: "Prompt-to-agent injection in GitHub Actions."
- Share demos with GitHub Actions, DevSecOps, and AI-agent communities.
- Submit to AI-agent security and GitHub Actions awesome lists.

## Success Signals

- Maintainers understand the `P2A` and `P2S` finding names without explanation.
- Users paste real workflow snippets into issues.
- Stars come from security engineers and maintainers, not only launch traffic.
