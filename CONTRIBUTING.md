# Contributing

Thanks for helping make AI-powered workflows safer.

The most useful contributions are small fixture-driven improvements:

1. Add a workflow fixture that shows the missed or noisy pattern.
2. Add or update a `node --test` case.
3. Keep the finding text concise and actionable.

`awi-scan` should stay focused on Agentic Workflow Injection. General GitHub Actions lint rules are better handled by tools like `actionlint` and `zizmor`.
